# TASK 0010: Eval — topological graph evaluator
Phase: 1
Crate: ailang-exec
Depends on: 0009 (NodeRegistry)

## Goal
Implement `eval(graph, registry)` — walk the graph in topological order, call
each node's registered function with its input values, and return the output
values of all nodes. This is the first complete execution path in ailang.

## Interface
```rust
// FILE: crates/ailang-exec/src/eval.rs
use crate::registry::{ExecError, NodeRegistry};
use crate::value::Value;
use ailang_core::graph::Graph;
use std::collections::HashMap;

/// Result of evaluating a graph: map from NodeIdx to that node's output Values.
pub type EvalResult = HashMap<usize, HashMap<String, Value>>;

#[derive(Debug, thiserror::Error)]
pub enum EvalError {
    #[error("cycle detected in graph")]
    Cycle,
    #[error("node {0}: {1}")]
    NodeFailed(usize, ExecError),
    #[error("missing output for edge from node {0} port {1}")]
    MissingOutput(usize, usize),
}

/// Evaluate the graph using the given registry.
/// Executes nodes in topological order (Kahn's algorithm).
/// Returns the outputs of every node.
pub fn eval(graph: &Graph, registry: &NodeRegistry) -> Result<EvalResult, EvalError>;
```

## Constraints
- Topological sort via Kahn's algorithm (in-degree table, queue of zero-in-degree nodes).
- A node is ready when all its input edges' source nodes have been executed.
- Pass each node its inputs by port name: for each incoming edge to dst_port,
  look up src_node's output by src_port name.
- Nodes with no inputs (sources) are executed first.
- Return `EvalError::Cycle` if topological sort doesn't process all nodes.
- No `unsafe`. No IO.

## Acceptance tests
```rust
// FILE: crates/ailang-exec/src/eval_tests.rs
#[cfg(test)]
mod tests {
    use ailang_core::{graph::{Graph, NodeDef, PortDef}, node_id::NodeId, ty::Type};
    use ailang_effects::EffectSet;
    use crate::{eval::eval, registry::NodeRegistry, value::Value};

    fn make_graph() -> (Graph, NodeRegistry) {
        let mut g = Graph::new();
        let mut r = NodeRegistry::new();

        // Source: Const node emitting Int(5)
        r.register_const("out", Value::Int(5));
        let src = g.add_node(NodeDef {
            id: NodeId::of(b"src"), kind: "Const:out".into(),
            inputs: vec![], effects: EffectSet::empty(),
            outputs: vec![PortDef { name: "out".into(), ty: Type::Int }],
        });

        // Sink: identity node that passes "in" → "out"
        r.register("identity", Box::new(|inputs| Ok(inputs)));
        let dst = g.add_node(NodeDef {
            id: NodeId::of(b"dst"), kind: "identity".into(),
            inputs: vec![PortDef { name: "out".into(), ty: Type::Int }],
            outputs: vec![PortDef { name: "out".into(), ty: Type::Int }],
            effects: EffectSet::empty(),
        });

        g.add_edge(src, 0, dst, 0).unwrap();
        (g, r)
    }

    #[test]
    fn simple_chain_evaluates() {
        let (g, r) = make_graph();
        let result = eval(&g, &r).unwrap();
        // dst node (idx 1) should have output "out" = Int(5)
        assert_eq!(result[&1]["out"], Value::Int(5));
    }

    #[test]
    fn empty_graph_evaluates() {
        let g = Graph::new();
        let r = NodeRegistry::new();
        let result = eval(&g, &r).unwrap();
        assert!(result.is_empty());
    }
}
```
